# Database Planning Project

This project was created as part of an exam on **database planning**. The report, written in Italian, analyzes a potential scenario and develops both a **conceptual** and **logical database plan**.  

Additionally, the project includes a set of the most commonly used queries for implementation.

## Project Structure

## Features

- Logical database design
- Common queries for implementation
